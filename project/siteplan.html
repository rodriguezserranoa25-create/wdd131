Website Planning Document

Site Name

GoalGear – Football T-Shirt Shop

I chose this name because it represents a brand that focuses on football-related apparel, especially replica and fan T-shirts. The name is easy to remember and instantly communicates the purpose of the website.

Site Purpose

The purpose of the site is to allow visitors to browse a selection of football T-shirts, view product details, and add items to a cart. The website will include dynamic JavaScript elements such as filtering, product detail pages, and a cart system stored in localStorage. A contact form will allow users to send messages.

Scenarios

  • “Where can I view all the football T-shirts available?”
  • “How do I add a shirt to my cart and check my total?”

Typography

Main Font: system-ui, -apple-system, "Segoe UI", Roboto, sans-serif

  • Headings: system-ui
  • Body text: system-ui
  • Buttons: system-ui (bold)

This font is clean, readable, and modern, matching a simple e-commerce look.
